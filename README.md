# Task Management Application

## Описание
Это веб-приложение для управления задачами, которое позволяет пользователям создавать, редактировать,
удалять и отслеживать статус задач. Приложение использует Django с Django REST Framework для бэкенда и предоставляет
API для взаимодействия с фронтендом.

## Функциональные возможности
- Регистрация и авторизация пользователей
- Создание, редактирование и изменение состояния задачи (выполнена / не выполнена)
- Отображение списка задач  по статусу
- Взаимодействие с API для выполнения операций с задачами

## Технологии
- Python 3.x
- Django
- Django REST Framework
- PostgreSQL
- Bootstrap (для фронтенда)
- Poetry (для управления зависимостями)

## Установка
1. Клонируйте репозиторий: https://github.com/kseniya161-max/DRF_technical/pull/3
2. poetry install
3. poetry shell
4. Скопируйте файл .env.sample в .env и заполните его необходимыми значениями.
5. python manage.py migrate
6. python manage.py runserver
7. poetry run python manage.py runserver
8. Перейдите на http://127.0.0.1:8000/api/tasks/ для доступа к приложению.
